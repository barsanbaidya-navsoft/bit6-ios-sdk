BXULocationViewController Class Reference

Inherits from UIViewController
Declared in BXULocationViewController.h

Overview

Convenience subclass of UIViewController to display a location in a MKMapView.

This viewController will present an UITabBar with options to open the location in AppleMaps, others applications can be configured as well using the BXULocationViewControllerDelegate protocol.
